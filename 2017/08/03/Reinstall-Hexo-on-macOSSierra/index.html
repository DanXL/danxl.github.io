利用Homebrew安装Git

同Node.js和NPM，安装Git就是一样的步骤了

同时，我们也需要将Git与GitHub账号建立SSH关联。

Step1 检查SSH Keys是否存在，如果存在则跳过Step 2

Step2 生成新的SSH Key

默认情况下，会在相应路径下（~/.ssh/id_rsa.pub）生成id_rsa和id_rsa.pub两个文件，其中前者是私钥，后者是公钥。私钥要保护好，公钥要上传到GitHub用于验证。

Step3 上传SSH Key

同Win7版本一样，将密钥复制到剪切板，然后打开登录GitHub -> Account Settings -> SSH keys and GPG keys -> New SSH key -> 填写并Add SSH key。
完成后测试一下
